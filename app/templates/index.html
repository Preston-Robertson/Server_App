Authentication

        

          The bearer token lives in /etc/gamesrv.env on the LXC
          (GAMESRV_TOKEN=..., mode 640 root:gamesrv).
          Paste it once and this browser will remember it in
          localStorage. Clear it with Forget token.
        

        

          
          
          
          
          
        

      


      

        
Manager Self-Update

        

          Pulls the latest commit on the repo's default branch, reinstalls
          pinned Python requirements into the venv, restarts the manager,
          health-checks /healthz, and automatically rolls back
          on failure. Full detail in update.sh / README.md.
