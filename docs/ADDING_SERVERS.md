# Adding a new server

## 1. Pick a `type`

- `minecraft-java` — Vanilla / Paper / Fabric / Forge (Java Edition).
- `steamcmd` — anything installable via anonymous SteamCMD (Palworld, Valheim,
  ARK, 7DTD, etc.). Needs `steam_app_id`.
- `custom` — you write `start.sh` yourself. Use this for Bedrock, PZ,
  Terraria, or anything not covered above.

## 2. Definition schema

`servers/<name>.yml`:

```yaml
name: myserver                       # [a-z][a-z0-9-]{1,31}, must match filename stem
type: minecraft-java                 # minecraft-java | steamcmd | custom
install_dir: /srv/gameservers/myserver
world_dir: /opt/gamesrv/worlds/myserver   # on the TrueNAS bind mount
port: 25565
memory_mb: 4096                      # -Xmx for Java; informational otherwise
java_args: ""                        # minecraft-java only
start_cmd: "./start.sh"              # regenerated by install for minecraft/steamcmd
stop_cmd: ""                         # console command sent before shutdown (minecraft: "stop")
auto_start_on_boot: true

# steamcmd only:
steam_app_id: 0
steam_beta: ""                       # optional public-beta branch name

extra_env: {}                        # extra KEY=VAL written to install_dir/server.env

rcon:
  enabled: false
  port: 25575
  password_env: ""                   # NAME of env var in /etc/gamesrv.env; never inline pw

backup:
  enabled: true
  target: ""                         # blank => world_dir
```

## 3. Workflow

1. Create the YAML (or use the dashboard's **+ New Server**).
2. **Control → Install** to provision.
3. Upload jars / configs (**Files** tab).
4. **Control → Start**.
5. **Control → Enable on Boot** once you're happy.

## 4. Adding a new `steamcmd` app not in the built-in recipes

Edit `app/types/steamcmd.py`, add to `_APP_RECIPES`:

```python
_APP_RECIPES[123456] = {
    "name": "mygame",
    "run": "./MyGameServer -port={port} -someflag",
    "saves_rel": "Saved/SaveGames",   # relative to install_dir; symlinked to world_dir
}
```

Restart the manager. Set `type: steamcmd`, `steam_app_id: 123456` in your YAML,
then **Install**.

## 5. Converting a `.bat` launcher

```bash
python3 /opt/gamesrv/scripts/convert_bat_to_sh.py /path/to/start.bat > /tmp/start.sh
# Review /tmp/start.sh, wrap the launch line in a tmux session, then upload as install/start.sh
```

Minimum tmux wrapper (for the console + graceful stop features to work):

```bash
#!/usr/bin/env bash
set -euo pipefail
cd "$(dirname "$0")"
tmux kill-session -t "gs-myserver" 2>/dev/null || true
exec tmux new-session -s "gs-myserver" -n game "./MyGameServer -port=12345"
```
